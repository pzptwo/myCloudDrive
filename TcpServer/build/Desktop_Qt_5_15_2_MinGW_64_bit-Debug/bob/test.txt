一、free -m
功能：查看系统内存使用情况
命令说明：free 用于显示内存信息，-m 表示以 MB 为单位展示。
输出字段：
total：总内存大小
used：已使用内存
free：空闲内存
buff/cache：系统缓存内存
available：应用程序可使用内存
实验作用：采集内存数据，用于客户端绘制内存饼图。
二、ps aux | wc -l
功能：统计系统当前总进程数量
命令说明：
ps aux：列出系统所有进程
|：管道符，将前一条命令结果传递给后一条命令
wc -l：统计输出内容的行数，即进程总数
实验作用：获取进程数量，用于客户端绘制进程数量变化曲线。
三、ip link | grep 'UP' | wc -l
功能：查看已启用的网络接口（网卡）数量
命令说明：
ip link：查看网卡信息
grep 'UP'：筛选状态为 UP（已启用）的网卡
wc -l：统计网卡数量
实验作用：采集网络接口使用数量，作为系统健康监控指标。
四、df -h /
功能：查看磁盘容量及使用情况
命令说明：
df：查看文件系统磁盘使用情况
-h：以 G、M 等易读单位显示
/：查看系统根分区（系统盘）信息
输出字段：
Size：磁盘总容量
Used：已使用容量
Avail：可用容量
Use%：磁盘使用率
实验作用：采集硬盘信息，用于客户端绘制磁盘使用饼图。
五、top -bn1 | grep 'Cpu'
功能：查看 CPU 使用率（系统健康指标）
命令说明：
top：系统实时监控工具
-bn1：仅采集 1 次数据后退出，适合程序调用
grep 'Cpu'：筛选出 CPU 相关信息行
输出字段：
us：用户态 CPU 占用率
sy：内核态 CPU 占用率
id：空闲 CPU 占比
实际 CPU 使用率 = 100 - id
实验作用：采集 CPU 使用率，展示系统负载健康状态。
